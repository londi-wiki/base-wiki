# Allgemein

$ 25 \bmod 11 = 3 $

$ -4 \bmod 11 = -4 + 11 = 7 $


# Schnelle Exponentiation

$$ 
5^{2 ^{ 2 } } \bmod 11 \\ 
5^{ 2 } \bmod 11 = 25 \bmod 11 = 3 \\
3^{ 2 } = 9
$$
