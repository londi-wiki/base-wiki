# Ollama

Ollama ist ein CLI Tool um bekannte LLM Modelle auf eine Weise auszuprobieren.

## Anleitung

1. [Ollama installieren](https://ollama.com/download)
2. Starten, z.B. llama 2: `ollama run llama2`
3. Mit `/bye` beenden
