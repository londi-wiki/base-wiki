# Musiktheorie für Ukulele – Eine Einführung

---

## 1. Die Notennamen

In der westlichen Musik gibt es **7 Grundtöne**, die mit Buchstaben benannt werden:

```
C – D – E – F – G – A – B (H)
```

> **Hinweis:** Im deutschen Sprachraum heisst der 7. Ton **H** (nicht B). Das englische **B** entspricht dem deutschen **B♭** (B-Dur mit Vorzeichen). Das kann am Anfang verwirrend sein – merke dir einfach: Englisch **B = Deutsch H**.

Diese 7 Töne wiederholen sich immer wieder – nach dem H/B kommt wieder das C, nur höher.

---

## 2. Halbtonschritte & Vorzeichen

Zwischen manchen Tönen gibt es **Zwischentöne** – diese heissen **Halbtöne**. Sie werden durch Vorzeichen markiert:

- **♯ (Kreuz / Sharp):** erhöht einen Ton um einen Halbton → z.B. C♯ (gesprochen: "Cis")
- **♭ (Be / Flat):** erniedrigt einen Ton um einen Halbton → z.B. B♭ (gesprochen: "B")

Die vollständige chromatische Tonleiter (alle 12 Halbtöne):

```
C – C♯/D♭ – D – D♯/E♭ – E – F – F♯/G♭ – G – G♯/A♭ – A – A♯/B♭ – H – C
```

**Wichtig:** E–F und H–C haben **keinen** Halbton dazwischen! Das sind natürliche Halbtöne.

### Alle 12 Töne mit Aussprache

| Notation     | Aussprache  | Engl. Notation | Hinweis                        |
|--------------|-------------|----------------|--------------------------------|
| C            | Ce          | C              |                                |
| C♯ / D♭     | Cis / Des   | C# / Db        | zwei Namen, selber Ton         |
| D            | De          | D              |                                |
| D♯ / E♭     | Dis / Es    | D# / Eb        | zwei Namen, selber Ton         |
| E            | E           | E              | kein Halbton zwischen E und F! |
| F            | Ef          | F              |                                |
| F♯ / G♭     | Fis / Ges   | F# / Gb        | zwei Namen, selber Ton         |
| G            | Ge          | G              |                                |
| G♯ / A♭     | Gis / As    | G# / Ab        | zwei Namen, selber Ton         |
| A            | A           | A              |                                |
| A♯ / B♭     | Ais / B     | A# / Bb        | deutsches "B" = englisches "Bb"|
| H            | Ha          | B              | kein Halbton zwischen H und C! |

> **Merkhilfe für die "-is" / "-es" Regel:**
> - Erhöhte Töne (♯) enden meist auf **-is** → Cis, Dis, Fis, Gis, Ais
> - Erniedrigte Töne (♭) enden meist auf **-es** → Des, Es, Ges, As, B (Ausnahmen: Es und As ohne "-es")

## 3. Die Oktave

Eine **Oktave** ist der Abstand von einem Ton zum nächsten Ton mit **demselben Namen** – aber höher oder tiefer.

Beispiel:
```
C (tief) ──── 12 Halbtöne ──── C (hoch)
```

Das hohe C klingt "gleich", aber doppelt so hoch. In der Physik: die Frequenz verdoppelt sich.

Auf der Ukulele umfasst der Griffbrett-Bereich meist **1–2 Oktaven** pro Saite.

---

## 4. Noten und Notenwerte

Noten beschreiben nicht nur **welcher** Ton gespielt wird, sondern auch **wie lange**.

| Symbol | Name           | Dauer (relativ) | Schläge (bei 4/4-Takt) |
|--------|----------------|-----------------|------------------------|
| 𝅝      | Ganze Note     | 1 ganzer Takt   | 4 Schläge              |
| 𝅗𝅥      | Halbe Note     | ½ Takt          | 2 Schläge              |
| ♩      | Viertel Note   | ¼ Takt          | 1 Schlag               |
| ♪      | Achtel Note    | ⅛ Takt          | ½ Schlag               |
| ♬      | Sechzehntel    | 1/16 Takt       | ¼ Schlag               |

**Eselsbrücke:** Jede Note ist halb so lang wie die vorherige:
```
Ganze → Halbe → Viertel → Achtel → Sechzehntel
  4   →   2   →    1    →  0.5  →    0.25   Schläge
```

### Punktierte Noten
Ein **Punkt** hinter einer Note verlängert sie um die **Hälfte** ihres Werts:
- Punktierte Halbe = 2 + 1 = **3 Schläge**
- Punktierte Viertel = 1 + 0.5 = **1.5 Schläge**

---

## 5. Takt und Taktart

Der **Takt** ist eine gleichmässige Einteilung der Zeit in Einheiten.

Die **Taktart** (z.B. 4/4) zeigt:
- **Obere Zahl:** wie viele Schläge pro Takt
- **Untere Zahl:** welche Note einen Schlag zählt (4 = Viertelnote)

Häufige Taktarten:

| Taktart | Name          | Charakter                     |
|---------|---------------|-------------------------------|
| 4/4     | Viervierteltakt | Standard, Rock, Pop          |
| 3/4     | Dreivierteltakt | Walzer, runder Fluss         |
| 6/8     | Sechsachteltakt | Wiegend, oft "2 grosse Schläge" |

## 6. Die Tonleiter (Skala)

Eine **Tonleiter** ist eine geordnete Folge von Tönen nach einem bestimmten **Muster von Ganz- und Halbtönen**.

### Dur-Tonleiter (Major Scale)

Das Muster: **G – G – H – G – G – G – H**
(G = Ganzton = 2 Halbtöne, H = Halbton = 1 Halbton)

Beispiel: **C-Dur**
```
C – D – E – F – G – A – H – C
  G   G   H   G   G   G   H
```

Dieses Muster klingt "hell" und "fröhlich".

### Moll-Tonleiter (Minor Scale / natürliches Moll)

Das Muster: **G – H – G – G – H – G – G**

Beispiel: **A-Moll**
```
A – H – C – D – E – F – G – A
  G   H   G   G   H   G   G
```

Moll klingt "dunkler" und "melancholischer".

---

## 7. Intervalle

Ein **Intervall** ist der Abstand zwischen zwei Tönen. Wichtig für Melodie und Akkorde.

| Intervall   | Halbtöne | Beispiel (ab C) |
|-------------|----------|-----------------|
| Prime       | 0        | C – C           |
| Sekunde     | 2        | C – D           |
| Terz (gross)| 4        | C – E           |
| Terz (klein)| 3        | C – E♭          |
| Quarte      | 5        | C – F           |
| Quinte      | 7        | C – G           |
| Sexte       | 9        | C – A           |
| Oktave      | 12       | C – C (hoch)    |

---

## 8. Akkorde

Ein **Akkord** ist das gleichzeitige Klingen von mindestens **3 Tönen**.

### Dur-Akkord (Major Chord)
Aufbau: **Grundton + grosse Terz (4 Halbtöne) + Quinte (7 Halbtöne)**

Beispiel: **C-Dur = C + E + G**

### Moll-Akkord (Minor Chord)
Aufbau: **Grundton + kleine Terz (3 Halbtöne) + Quinte (7 Halbtöne)**

Beispiel: **A-Moll = A + C + E**

### Auf der Ukulele
Die Standard-Stimmung der Ukulele ist: **G – C – E – A** (von oben nach unten).

Das bedeutet, ein leerer Griff (alle offenen Saiten) ergibt Töne aus dem **C-Dur-Akkord**!

---

## 9. Zusammenfassung: Der grosse Überblick

```
TÖNE:       C  C♯  D  D♯  E  F  F♯  G  G♯  A  A♯  H  C
            └──────────── 12 Halbtöne = 1 Oktave ──────────┘

NOTENWERTE: Ganze → Halbe → Viertel → Achtel → Sechzehntel

TAKT:       4/4 = 4 Viertelschläge pro Takt

SKALEN:     Dur   = hell,  G–G–H–G–G–G–H
            Moll  = dunkel, G–H–G–G–H–G–G

AKKORDE:    Dur   = Grundton + grosse Terz + Quinte
            Moll  = Grundton + kleine Terz + Quinte

UKULELE:    Stimmung G – C – E – A
```
